# CSS-frameworks

This project is a responsive social media web app built as part of the CSS Frameworks course assignment. It includes an authentication page, a feed page with post functionality, and a profile page. The UI is styled using Bootstrap, and basic JavaScript is used for the mobile navigation menu.

## Features

- Responsive layout using Bootstrap.
- Authentication form with HTML validation.
- Mobile-friendly navigation menu (with JavaScript toggle).
- Feed page with post creation form, search, and sort options.
- Profile page with user info, follow button, and user posts.

## Prerequisites

- Node.js (v20+)
- npm

## Getting Started

### Installation

```bash
npm install
```

### Running the project

```bash
npm run dev
```

### Production build

```bash
npm run build
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production

## Technologies

- JavaScript (for mobile menu toggling)
- HTML
- CSS (including Bootstrap)
